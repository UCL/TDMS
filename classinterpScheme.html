double interpScheme::interpolate ( const double *  v,
const int  offset = 0 
) const

Executes the interpolation scheme on the data provided.

The interpolation schemes are all of the form interpolated_value = \sum_{i=first_nonzero_coeff}^{last_nonzero_coeff} scheme_coeffs[i] * v[i], so provided that the coefficients have been set correctly in construction (and the data gathered appropriately), we can run the same for loop for each interpolation scheme.

Parameters
vSample datapoints to use in interpolation
offset[Default 0] Read buffer from v[offset] rather than v[0]
Returns
double Interpolated value
