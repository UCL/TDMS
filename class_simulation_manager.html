Detailed Description

Manages the physics of TDMS and the simulation loop itself.

Takes in the contents of the input file and produces the output in the outputs member, so that it can be written to the specified output file.

Attributes are private to avoid external alterations to their content and potential reassignment of MATLAB pointers, resulting in leaking memory. The outputs can be accessed through a fetch-method within main.cpp if necessary.

Dispersive parameter. = inputs.matched_layer.kappa.{x,y,z}

NOTE: if dealing with Ksource, then the index used to fetch the gamma parameter is k = inputs.K1.index or inputs.K0.index, depending on whether we are updating in the zero_plnae or not. If dealing with Isource or Jsource, cell_c (which = the k index) is the index for the gamma parameter. See https://github.com/UCL/TDMS/issues/225

Dispersive parameter. Pulls values from inputs.matched_layer.gamma

Value of the source term in Yee cell (cell_a, cell_b, cell_c)

The split-field cell index to update

Field updates are signed based on the component, plane, and axis. [0] = E_s sign, [1] = J_c sign, [2] = J_s sign

This is the index that is to be passed to the (relevent) Source instance, to recover the source value at the relevant Yee cell.

cell_b and cell_c need to be offset by the corresponding {IJK}0.index before we can retrieve the source value.

◆ linear_ramp()

double SimulationManager::linear_ramp ( double  t)
inlineprivate

Evaluates the linear ramp function r(t)

r(t) = 1 if t > ramp_width * period = t / (ramp_width * period) if t <= ramp_width * period

Period = 1. / (omega_an / (2 * pi)) = 2 * pi / omega_an

NB Note that r(t) = min(1, t / (ramp_width * period))

Parameters
tTime to evaluate function at
Returns
double
