15 /**
16  * A generic grid entity. For example:
17  *
18  * ______________________ (2, 1, 0)
19  * / / /
20  * / / /
21  * /___________/_________/
22  * (0, 0, 0) (1, 0, 0) (2, 0, 0)
23  *
24  * has I_tot = 2, J_tot = 1, K_tot = 0.
25  *
26  * NOTE: For storage purposes, this means that field values associated to cells are stored _to the left_.
27  * That is, Grid(0,0,0) is associated to the cell (-1,-1,-1). This is contrary to the way values are associated to cells, where cell (0,0,0) is associated to the field values (0,0,0).
28  */
